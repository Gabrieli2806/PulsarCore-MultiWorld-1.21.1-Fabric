# Quantum

Quantum multi-world.

![Platform](https://img.shields.io/badge/platform-Fabric-informational) ![Minecraft](https://img.shields.io/badge/minecraft-1.21.1-green)

## How to Use

1. Download the `.jar` from [Releases](https://github.com/Gabrieli2806/PulsarCore-MultiWorld-1.21.1-Fabric/releases).
2. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft **1.21.1**.
3. Place the mod `.jar` in your `.minecraft/mods/` folder.
4. Launch Minecraft with the Fabric profile.
